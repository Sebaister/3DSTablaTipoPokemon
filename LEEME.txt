=== Visor de Tablas de Tipos Pokémon para Nintendo 3DS ===

Propósito:
Esta página web fue creada exclusivamente para ver tablas de tipos Pokémon directamente desde el navegador de la Nintendo 3DS, aprovechando funciones únicas de la consola:

Ventajas clave:
- [Minimizar con HOME] Presiona HOME para pausar y consultar tu juego, luego retoma al instante
- [Autosuficiente] No requiere teléfono inteligente externo
- [Optimizado para 3DS] Diseñado para las limitaciones del navegador:
  - Sin dependencias de JavaScript
  - Compatible solo con HTTP
  - Assets de baja resolución
  - Diseño de una columna

Notas técnicas:
- Imágenes comprimidas (<1MB cada una)
- Solo HTML/CSS básico (compatible con NetFront)
- Sin rastreo, anuncios o dependencias externas

Uso:
1. Abre en el navegador 3DS via http:// (no https://)
2. Usa el D-pad para navegar entre tablas
3. Presiona HOME cuando necesites minimizar

No es un producto oficial de Nintendo/Pokémon. Las tablas son solo de referencia.
